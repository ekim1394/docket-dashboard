# Notebooks

Interactive Jupyter notebooks for analyzing federal regulatory data from the Spicy Regs Dashboard.

## Available Notebooks

- **[2023_peak_analysis.ipynb](2023_peak_analysis.ipynb)** - Investigates why 2023 saw a record 50,279 federal dockets and 386 million public comments during the Biden administration.

## Run in the Cloud

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/ekim1394/docket-dashboard/main?labpath=notebooks%2F2023_peak_analysis.ipynb)

Click the badge above to launch the notebooks in your browser via [mybinder.org](https://mybinder.org) -- no local setup required.

```bash
pip install pandas matplotlib duckdb
```
